# Demo Project


## Live URL: https://demo-project.herokuapp.com


## Quick Start

  Clone the repository:

```bash
$ git clone https://github.com/amitdubey8888/demo-project.git
```

  Goto project directory:

```bash
$ cd demo-project
```

  Install dependencies:

```bash
$ npm install
```

  Start the app:

```bash
$ npm start
```

  View the app at: http://localhost:3000
